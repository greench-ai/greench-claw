# Plugin inventory

This page is generated from `extensions/*/package.json`, `GreenchClaw.plugin.json`,
and the root npm package `files` exclusions. Regenerate it with:

```bash
pnpm plugins:inventory:gen
```

## Definitions

- **Core npm package:** built into the `GreenchClaw` npm package and available without a separate plugin install.
- **Official external package:** GreenchClaw-maintained plugin omitted from the core npm package, kept in this official inventory, and installed on demand through ClawHub and/or npm.
- **Source checkout only:** repo-local plugin omitted from published npm artifacts and not advertised as an installable package.

Source checkouts are different from npm installs: after `pnpm install`, bundled
plugins load from `extensions/<id>` so local edits and package-local workspace
dependencies are available.

## Install a plugin

Use the **Distribution** column to decide whether install is needed. Plugins that
say `included in GreenchClaw` are already present in the core package. Official
external packages need one install, then a Gateway restart.

For example, Discord is an official external package:

```bash
GreenchClaw plugins install @GreenchClaw/discord
GreenchClaw gateway restart
GreenchClaw plugins inspect discord --runtime --json
```

Bare package specs try ClawHub first, then npm fallback. To force a source, use
`clawhub:@GreenchClaw/discord` or `npm:@GreenchClaw/discord`. After install, follow
the plugin's setup doc, such as [Discord](/channels/discord), to add credentials
and channel config. See [Manage plugins](/plugins/manage-plugins) for update,
uninstall, and publishing commands.
